### Clion
Download [This Repo](https://github.com/vulcan-dev/premake-cmake)  
Go to C:\Users\YourName\
Make a folder called **.premake**  
Copy the contents of **premake-cmake** (the zip you downloaded) into that folder

```
.\premake5.exe cmake
.\premake5.exe clion
```
